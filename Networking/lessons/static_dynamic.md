Static Routing

Definition: You manually type the routes into the routing table. The paths are fixed and do not change unless a human manually edits them.

How it works: The administrator tells the router: "To reach Network B, always go through Gateway X."

Best for: Small, simple networks where the layout never changes.

Dynamic Routing

Definition: Routers use special software (protocols) to talk to each other and "learn" the network layout automatically. They update their own routing tables in real-time.

How it works: Routers share information like: "Hey, I found a faster way to Network C" or "Road A is closed, everyone use Road B instead."

Best for: Large, complex networks (like the Internet or AWS) where links might go down or change frequently.
