1. Simple Definition

The /etc/hosts file is a local text file on your computer that maps hostnames to IP addresses.

It is the first place your computer looks before it ever goes out to the "Public Phonebook" (DNS).

2. The Hierarchy of "Checking"
   
When you type google.com in your browser, your computer follows this priority:

  Step 1: Check the /etc/hosts file (Your personal notes).
  
  Step 2: If it's not there, go to the DNS Server (The internet's phonebook).
