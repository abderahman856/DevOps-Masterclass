**Lesson 1 — What is Linux?**

Linux is:

  An operating system
  
  Used in servers
  
  Used in cloud (AWS, Azure, GCP)
  
  Used in DevOps pipelines

DevOps engineers use Linux because:
  
  Most servers run Linux
  
  It is stable and powerful
  
  It supports automation

**Lesson 2 — The Terminal**

The terminal is where you type commands.

Example:
    
    pwd

Shows:

    /home/user

It means:

  “This is where I am right now.”

**Lesson 3 — Navigation Commands**

See where you are

    pwd

List files

    ls
    ls -la
    ls -l
    ls -R

Move between folders

    cd folder
    cd ..
    cd ~

 DevOps Tip: You must move confidently inside servers.
